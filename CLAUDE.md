# Project: Pencil Design-to-Code
## Tech Stack
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Design Tool: Pencil.dev (`design.pen`)

## Instructions
- When I ask to "sync design," use the Pencil MCP server to read `design.pen`.
- Map Pencil frames to React components in `@/components`.
- Use Tailwind CSS for all styling.
- Maintain a strict 8px spacing grid.
